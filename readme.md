# 5. 스타일 시트 적용하는 3가지 방법

- 문서 내부 (style 태그)
- 문서 외부 별도로(css 파일 임포트)
- 인라인 스타일 (태그에 직접 스타일 지정)

## 5.1 문서내부

style 태그 내부

```html
<style>여기에</style>
```

## 5.2 인라인 스타일

태그의 style 속성에 직접 지정

```html
<p style="background-color: black;">
뭔가있음
</p>
```

## 5.3. 문서 외부에

